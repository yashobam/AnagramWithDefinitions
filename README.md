# AnagramWithDefinitions
All possible anagrams of words with definitions

# How to run it:
*Make sure that you're in the correct folder*
```bash
$ python AWD.py "WORD"
```
# Example:
```bash
$ python AWD.py leephant
```
Output:
```
Elephant
1) five-toed pachyderm
2) the symbol of the Republican Party
```
